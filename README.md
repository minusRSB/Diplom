## Кратко о проекте

В этом ноутбуке проведён разведочный анализ данных и построена модель машинного обучения для прогнозирования показателя Gprof на основе экономических и долевых характеристик.
Проект выполнен в формате end-to-end: от первичного анализа распределений и взаимосвязей до обучения и оценки модели.

Цель работы — показать аналитическое мышление, умение работать с данными и базовый DS-пайплайн, а не добиться максимального качества модели любой ценой.

## Данные

Используется табличный датасет training_data.csv, содержащий следующие признаки:

- Mrash, rash — показатели, связанные с расходами
- Mryn, ryn — показатели, связанные с рынком
- dol_low, dol_high — долевые характеристики
- Gprof — целевая переменная (показатель прибыли)

Перед анализом данные загружаются и проверяются на базовую корректность.

## Разведочный анализ данных (EDA)

В ноутбуке выполнено:

- анализ распределения целевой переменной Gprof,
- исследование корреляций между признаками,
- визуальный анализ связей Gprof с ключевыми факторами,
- сравнение распределений признака до и после стандартизации.

Графики сопровождаются интерпретациями и используются для понимания структуры данных и возможных зависимостей.

## Модель

Для прогнозирования Gprof используется:

- стандартизация признаков (StandardScaler),
- регрессионная модель MLPRegressor,
- разбиение данных на обучающую и тестовую выборки.

Качество модели оценивается с помощью MSE на тестовой выборке.
Модель и скейлер сохраняются для дальнейшего использования.
